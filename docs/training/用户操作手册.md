# 宁夏CRM作战地图 — 用户操作手册

> 本文档面向最终用户，详细介绍系统各模块的操作方法和功能说明。

---

## 一、系统登录和退出

### 1.1 登录系统

1. 打开浏览器，输入系统地址：`https://crm.example.com`
2. 在登录页面输入您的账号和密码
3. 点击【登录】按钮

> 首次登录后，系统会提示您修改初始密码。建议使用8位以上包含字母和数字的密码。

[截图: 登录页面]

### 1.2 退出系统

- 点击页面右上角用户名旁边的头像
- 选择【退出登录】
- 或使用快捷键：无（防止误操作）

> **提示**：长时间离开工位时，请务必退出系统，保护数据安全。

### 1.3 修改密码

1. 点击右上角头像 → 【个人设置】
2. 选择【修改密码】标签
3. 输入原密码、新密码和确认密码
4. 点击【保存】

[截图: 个人设置页面]

## 二、首页工作台概览

### 2.1 页面布局

[截图: 首页工作台]

首页工作台包含以下区域：

| 区域 | 位置 | 功能说明 |
|------|------|----------|
| 顶部导航栏 | 页面最上方 | 系统Logo、消息通知、用户头像 |
| 左侧菜单 | 页面左侧 | 6大模块入口 |
| KPI卡片区 | 首页上方 | 6个关键指标快速展示 |
| 优先事项区 | 首页左侧 | 待办提醒和告警 |
| 项目进度区 | 首页中部 | 本周项目进度跟踪 |
| 动态区 | 首页右侧 | 最新运维和商机动态 |

### 2.2 KPI卡片

首页展示6个核心业务指标：

| KPI名称 | 说明 | 数据来源 |
|---------|------|----------|
| 客户总数 | 系统中累计客户数量 | 客户资产模块 |
| 在途项目 | 当前进行中的项目数 | 项目交付模块 |
| 本月商机 | 本月新增商机数量 | 商机营销模块 |
| 待办工单 | 未完成的运维工单数 | 运维管理模块 |
| 本月拜访 | 本月计划拜访次数 | 商机营销模块 |
| 知识素材 | 累计知识素材数量 | 知识分享模块 |

> 点击任意KPI卡片，可跳转至对应模块的详情页面。

### 2.3 优先事项

显示当前用户需要处理的待办事项：

- 待跟进的商机
- 即将到期的项目节点
- 待处理的运维工单
- 待确认的会议邀请

> 优先事项按紧急程度排序，红色标记为紧急事项，黄色标记为重要事项。

### 2.4 项目进度

展示本周的项目进展情况：

| 项目状态 | 颜色标识 | 说明 |
|----------|----------|------|
| 未开始 | 灰色 | 项目尚未启动 |
| 进行中 | 蓝色 | 项目正常推进 |
| 有风险 | 橙色 | 项目存在延期或风险 |
| 已完成 | 绿色 | 项目已交付 |

> 点击项目可查看详细进度和时间线。

### 2.5 动态

实时展示系统内的最新动态：

- 新创建的运维工单
- 新录入的商机
- 新发布的知识素材
- 项目节点状态变更

## 四、项目交付模块

### 4.1 项目概览

[截图: 项目概览页面]

项目概览提供项目列表和统计视图：

| 功能 | 说明 |
|------|------|
| 项目列表 | 分页展示所有项目，支持筛选和排序 |
| 筛选条件 | 状态、类型、负责人、时间范围 |
| 搜索 | 按项目名称或客户名称搜索 |
| 统计卡片 | 项目总数、进行中、有风险、已完成 |
| 快速操作 | 新建项目、导出列表 |

### 4.2 新建项目

**操作步骤**：

1. 点击【项目交付】→【项目概览】
2. 点击右上角【新建项目】按钮
3. 填写项目基本信息：
   - 项目名称（必填）
   - 客户名称（必填，从客户档案选择）
   - 项目类型（必填）
   - 负责人（必填）
   - 计划开始/结束日期
   - 项目描述
4. 点击【保存】

[截图: 新建项目表单]

> 项目创建成功后，系统会自动创建默认的项目节点时间线。

### 4.3 项目节点跟踪

[截图: 项目节点页面]

项目节点页面以时间线形式展示项目各阶段：

| 节点状态 | 图标 | 说明 |
|----------|------|------|
| 待开始 | 灰色圆点 | 节点尚未启动 |
| 进行中 | 蓝色圆点 | 节点正在执行 |
| 已完成 | 绿色圆点 | 节点已交付 |
| 已延期 | 红色圆点 | 节点超过计划日期 |

**操作功能**：

- 添加节点：点击【添加节点】，填写节点名称、计划日期和负责人
- 更新状态：点击节点，选择新的状态
- 删除节点：点击节点旁的删除按钮（仅未开始节点可删除）

### 4.4 会议管理

[截图: 会议管理页面]

会议管理用于记录项目相关的会议信息：

**新建会议**：

1. 选择项目
2. 点击【新建会议】
3. 填写会议信息：
   - 会议主题
   - 会议时间
   - 参与人员
   - 会议地点/链接
   - 会议内容
4. 点击【保存】

**会议记录**：

- 查看历史会议列表
- 编辑会议记录
- 上传会议纪要附件

### 4.5 交付看板

[截图: 交付看板页面]

交付看板采用 Kanban（看板）模式展示项目状态：

| 列名 | 说明 |
|------|------|
| 待启动 | 已创建但未开始的项目 |
| 进行中 | 正在执行的项目 |
| 待验收 | 已完成等待客户验收的项目 |
| 已完成 | 已验收交付的项目 |

**操作**：

- 拖拽项目卡片在不同列之间移动
- 点击卡片查看项目详情
- 双击卡片编辑项目信息

> 看板视图实时同步，团队成员可看到最新的项目状态变化。

## 五、客户资产模块

### 5.1 客户档案

[截图: 客户档案页面]

客户档案是系统中最核心的模块，提供完整的客户信息管理：

**客户列表**：

| 功能 | 说明 |
|------|------|
| 列表展示 | 客户名称、区域、行业、负责人、状态 |
| 筛选 | 按区域、行业、状态、负责人筛选 |
| 搜索 | 按客户名称、联系人搜索 |
| 排序 | 按名称、创建时间、更新时间排序 |
| 分页 | 每页10/20/50条 |

**新建客户**：

1. 点击【客户资产】→【客户档案】
2. 点击【新建客户】
3. 填写客户信息：
   - 基本信息：名称、简称、区域、行业、地址
   - 联系信息：电话、邮箱、官网
   - 关系人：姓名、职务、联系方式
   - 其他信息：客户等级、合作状态、备注
4. 点击【保存】

[截图: 新建客户表单]

> **数据录入规范**：客户名称请使用营业执照全称，避免使用简称。

**客户详情**：

点击客户名称进入详情页，查看：

- 客户基本信息
- 关系人列表
- 关联项目
- 关联商机
- 历史拜访记录
- 变更历史

### 5.2 区域地图

[截图: 区域地图页面]

区域地图以可视化方式展示客户地理分布：

- 地图缩放：鼠标滚轮或点击+/-按钮
- 客户标记：点击标记查看客户信息
- 区域筛选：选择特定区域聚焦
- 数据统计：各区域客户数量分布

> 地图数据基于客户地址信息，请确保地址准确以正确显示位置。

### 5.3 关系网络

[截图: 关系网络页面]

关系网络展示客户之间的关系图谱：

- 节点：客户或关系人
- 连线：业务关系或人际关系
- 点击节点：查看详细信息
- 拖拽：调整布局

> 关系网络有助于发现潜在的业务机会和决策路径。

## 六、运维管理模块

### 6.1 运维工单

[截图: 运维工单页面]

运维工单用于管理日常运维任务：

**工单列表**：

| 字段 | 说明 |
|------|------|
| 工单编号 | 系统自动生成，如 WO-20260101-001 |
| 工单类型 | 故障处理、巡检、变更、其他 |
| 优先级 | 紧急、高、中、低 |
| 状态 | 待处理、处理中、待确认、已完成 |
| 负责人 | 工单处理人 |
| 创建时间 | 工单创建时间 |

**新建工单**：

1. 点击【运维管理】→【运维工单】
2. 点击【新建工单】
3. 填写工单信息：
   - 工单类型
   - 优先级
   - 客户/设备关联
   - 问题描述
   - 期望完成时间
4. 点击【提交】

**工单处理流程**：

```
待处理 → 处理中 → 待确认 → 已完成
  ↑__________↓______↑
         可退回
```

### 6.2 巡检计划

[截图: 巡检计划页面]

巡检计划用于安排定期设备巡检：

- 查看巡检计划日历
- 创建周期性巡检任务
- 记录巡检结果
- 查看巡检历史

**创建巡检计划**：

1. 点击【新建巡检计划】
2. 选择巡检对象（客户/设备）
3. 设置巡检频率（每日/每周/每月）
4. 设置巡检内容和检查项
5. 分配巡检负责人
6. 点击【保存】

### 6.3 资产台账

[截图: 资产台账页面]

资产台账管理客户现场的设备资产：

| 字段 | 说明 |
|------|------|
| 资产编号 | 系统自动生成或手动录入 |
| 资产名称 | 设备名称 |
| 资产类型 | 服务器、网络设备、安全设备等 |
| 所属客户 | 资产所在客户 |
| 部署位置 | 具体地址/机房 |
| 序列号 | 厂商序列号 |
| 保修到期日 | 厂商保修截止日期 |
| 状态 | 运行中、已停用、已报废 |

**资产操作**：

- 新增资产：填写资产信息
- 编辑资产：修改资产信息
- 删除资产：标记资产为停用（不会物理删除）
- 导出资产：导出资产清单

[截图: 拜访计划页面]

4. 填写拜访目的

2. 点击【签到】按钮
3. 系统记录签到时间和位置（需授权定位）

## 九、常用操作

### 9.1 搜索

系统提供全局搜索功能：

1. 点击顶部导航栏的搜索框
2. 输入关键词（客户名称、项目名称、工单编号等）
3. 按回车或点击搜索按钮
4. 查看搜索结果，按类型筛选

> 搜索支持模糊匹配，可输入关键词的一部分进行搜索。

### 9.2 筛选

列表页面支持多条件筛选：

1. 点击列表上方的【筛选】按钮
2. 选择筛选条件（如状态、时间范围、负责人等）
3. 点击【应用】
4. 列表自动刷新显示筛选结果
5. 点击【重置】清除所有筛选条件

### 9.3 导出

系统支持多种导出格式：

| 导出格式 | 适用场景 | 文件扩展名 |
|----------|----------|------------|
| Excel | 数据分析和二次处理 | .xlsx |
| CSV | 数据导入其他系统 | .csv |
| PDF | 打印和归档 | .pdf |

### 9.4 消息通知

系统通过以下方式发送通知：

| 通知类型 | 触发场景 | 查看方式 |
|----------|----------|----------|
| 待办提醒 | 有未处理工单/商机 | 首页优先事项区 |
| 节点提醒 | 项目节点到期 | 消息中心+首页 |
| 会议提醒 | 即将开始的会议 | 消息中心+邮件 |
| 系统通知 | 系统公告 | 消息中心 |
| 实时推送 | 新工单/商机 | WebSocket弹窗 |

**消息中心**：

1. 点击顶部导航栏的铃铛图标
2. 查看未读消息列表

## 十、常见问题FAQ

### Q1：忘记密码怎么办？

**A**：请联系系统管理员重置密码。管理员可在【用户管理】中为您重置密码，重置后的默认密码会发送到您的邮箱。

### Q2：登录时提示"账号已被禁用"？

**A**：请联系管理员确认账号状态。如账号被误禁用，管理员可在【用户管理】中重新启用。

### Q3：为什么有些页面看不到或操作按钮是灰色的？

**A**：这是权限控制导致的。您的角色可能没有该页面或功能的访问权限。如需权限调整，请联系管理员。

### Q4：导出的数据文件打不开？

**A**：请使用Microsoft Office 2016及以上版本或WPS Office打开。如仍无法打开，请检查文件是否完整下载。

### Q5：客户地图显示位置不正确？

**A**：请检查客户地址信息是否准确。地图基于地址解析，如地址模糊或不完整可能导致定位偏差。

### Q6：如何修改已创建的项目信息？

**A**：进入【项目交付】→【项目概览】，找到目标项目，点击【编辑】按钮即可修改。部分字段（如项目编号）不可修改。

### Q7：商机阶段可以回退吗？

**A**：可以。在商机详情页，点击当前阶段下拉框，选择上一阶段即可。但已成交的商机不能回退到成交前的阶段。

### Q8：为什么收不到系统通知？

**A**：请检查：1）浏览器是否开启了弹窗拦截；2）系统设置中是否关闭了通知；3）网络连接是否正常。如问题持续，请联系技术支持。

### Q9：可以同时登录多个设备吗？

**A**：可以，但出于安全考虑，同一账号最多同时登录3台设备。超过后将自动踢出最早登录的设备。

### Q10：如何提交系统问题反馈？

**A**：您可以通过以下方式反馈：1）点击页面右下角的【意见反馈】按钮；2）发送邮件至技术支持邮箱；3）联系系统管理员。请尽量描述问题发生的具体操作步骤和错误提示。
