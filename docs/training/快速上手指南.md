# 宁夏CRM作战地图 — 快速上手指南

> 欢迎加入宁夏CRM作战地图！本指南帮助新用户在第一天快速上手，掌握系统核心操作。

---

## 一、新用户第一天操作清单（10步）

### Step 1：登录系统

打开浏览器，访问 `https://crm.example.com`，使用管理员分配的账号密码登录。

> 首次登录后，系统会强制要求修改密码。请设置8位以上包含大小写字母和数字的密码。

[截图: 登录页面]

---

### Step 2：熟悉首页布局

登录后，您将看到首页工作台。花5分钟浏览以下区域：

- [ ] 顶部导航栏：Logo、消息、用户头像
- [ ] 左侧菜单：6大模块入口
- [ ] KPI卡片：6个关键业务指标
- [ ] 优先事项：您的待办任务
- [ ] 项目进度：本周项目概览
- [ ] 动态区：最新业务动态

[截图: 首页工作台]

> **提示**：点击左侧菜单的各个模块，了解系统包含哪些功能。

---

### Step 3：完善个人资料

1. 点击右上角头像 → 【个人设置】
2. 上传个人头像（建议使用职业照）
3. 确认姓名、手机号、邮箱信息准确
4. 点击【保存】

[截图: 个人设置页面]

---

### Step 4：了解您的权限

不同的角色拥有不同的功能权限：

| 角色 | 主要权限 | 典型用户 |
|------|----------|----------|
| 管理员 | 全部功能 | 系统管理员 |
| 销售经理 | 客户管理+商机+报表 | 销售主管 |
| 销售代表 | 客户+商机+拜访 | 一线销售 |
| 项目经理 | 项目+客户+运维 | 项目主管 |
| 运维工程师 | 运维工单+资产 | 运维人员 |
| 普通用户 | 查看+基础操作 | 新员工/访客 |

> 如有权限疑问，请联系管理员确认您的角色配置。

---

### Step 5：查看并处理待办

在首页【优先事项】区域：

- 查看是否有分配给您的任务
- 如有紧急事项（红色标记），优先处理
- 点击待办可直接跳转处理页面

[截图: 优先事项区域]

---

### Step 6：浏览客户列表

1. 点击左侧菜单【客户资产】→【客户档案】
2. 浏览您负责的客户列表
3. 点击客户名称查看详情
4. 了解客户的基本信息、关系人和关联项目

[截图: 客户档案页面]

> 客户档案是系统的核心数据，请确保您了解每个客户的基本情况。

---

### Step 7：查看您的项目

1. 点击左侧菜单【项目交付】→【项目概览】
2. 查看您参与或负责的项目
3. 了解项目当前状态和进度
4. 点击项目查看节点时间线

[截图: 项目概览页面]

---

### Step 8：学习录入一条商机

1. 点击【商机营销】→【商机池】
2. 点击【新建商机】
3. 填写商机信息：
   - 商机名称：简短描述机会
   - 客户：从客户档案选择
   - 预计金额：预估成交金额
   - 预计成交日期：计划签约时间
4. 点击【保存】

[截图: 新建商机表单]

> 商机录入是销售工作的基础，请确保信息准确完整。

---

### Step 9：体验消息通知

1. 点击顶部导航栏的铃铛图标
2. 查看系统通知和待办提醒
3. 点击【全部已读】清空未读
4. 了解消息通知的获取方式

[截图: 消息中心]

---

### Step 10：收藏本指南

将本快速上手指南保存到您的收藏夹，方便随时查阅。

完成以上10步，您已基本掌握系统的核心操作！接下来，请参考【每日必做】和【每周必做】建立日常工作习惯。

---

## 二、每日必做（5件事）

养成每天登录系统完成以下5件事的习惯：

### 1. 登录并查看待办（5分钟）

- 登录系统
- 查看首页【优先事项】
- 处理紧急待办（红色标记）

### 2. 更新商机状态（10分钟）

- 进入【商机营销】→【商机池】
- 查看您负责的商机
- 更新有进展的商机阶段
- 添加跟进记录

### 3. 检查项目进度（10分钟）

- 进入【项目交付】→【项目概览】
- 查看负责项目的当前状态
- 更新已完成的节点
- 如有延期风险，更新项目状态为"有风险"

### 4. 处理运维工单（如适用，10分钟）

- 进入【运维管理】→【运维工单】
- 查看分配给您的工单
- 处理待办工单，更新状态

### 5. 查看动态和通知（5分钟）

- 查看首页【动态】区域
- 阅读系统通知
- 确认消息已读

> **每日总计约40分钟**，建议分上午、下午两次完成，保持数据实时更新。

---

## 三、每周必做（3件事）

### 1. 周报数据整理（周一上午，30分钟）

- 导出本周的客户、项目、商机数据
- 检查数据完整性
- 确认上周承诺事项的完成情况

### 2. 拜访计划安排（周一下午，20分钟）

- 进入【商机营销】→【拜访计划】
- 创建本周的拜访计划
- 确认拜访客户、时间和目的
- 通知相关同事

### 3. 知识素材学习（周五下午，20分钟）

- 进入【知识分享】→【知识素材】
- 浏览本周新上传的素材
- 下载需要的文档
- 给有价值的素材点赞

> **每周总计约70分钟**，建议在周一集中完成前两件事，周五完成学习事项。

---

## 四、每季度必做（1件事）

### 季度数据回顾（季度末最后一周，2小时）

1. 导出本季度的完整业务数据：
   - 客户新增/流失统计
   - 项目完成/延期统计
   - 商机转化报表
   - 运维工单统计

2. 分析数据，总结：
   - 本季度业绩达成情况
   - 客户满意度趋势
   - 项目交付质量
   - 需要改进的环节

3. 更新客户档案：
   - 检查客户信息是否过时
   - 更新联系人信息
   - 确认客户状态（活跃/休眠/流失）

4. 制定下季度计划：
   - 重点客户清单
   - 目标商机清单
   - 项目交付计划

> 季度回顾是提升业务管理水平的关键环节，建议预留充足时间认真完成。

## 五、数据录入规范

### 5.1 客户数据录入规范

| 字段 | 规范要求 | 示例 |
|------|----------|------|
| 客户名称 | 使用营业执照全称 | 宁夏某某科技有限公司 |
| 客户简称 | 2-6字简称 | 某某科技 |
| 区域 | 选择标准行政区划 | 银川市金凤区 |
| 行业 | 选择标准行业分类 | 信息技术 |
| 地址 | 详细到门牌号 | 银川市金凤区宁安大街XX号XX大厦12层 |
| 电话 | 包含区号 | 0951-XXXXXXXX |
| 联系人 | 真实姓名 | 张三 |
| 邮箱 | 有效邮箱地址 | zhangsan@example.com |

> 客户数据是系统的基础，请确保录入信息准确、完整。不规范的数据将影响报表统计和地图定位。

### 5.2 项目数据录入规范

| 字段 | 规范要求 | 说明 |
|------|----------|------|
| 项目名称 | 客户简称+项目类型 | 某某科技-数据中心建设 |
| 项目类型 | 选择标准类型 | 新建/改造/维护 |
| 计划日期 | 实际业务计划日期 | 影响进度统计 |
| 负责人 | 主负责人 | 唯一主负责人 |
| 项目描述 | 100字以内 | 简明描述项目内容 |
| 节点日期 | 计划完成日期 | 用于时间线跟踪 |

### 5.3 工单数据录入规范

| 字段 | 规范要求 | 说明 |
|------|----------|------|
| 工单类型 | 选择正确类型 | 影响统计分类 |
| 优先级 | 客观评估 | 紧急=业务中断，高=重大影响 |
| 问题描述 | 详细描述问题 | 包含现象、时间、影响范围 |
| 处理记录 | 按时间记录 | 每次处理更新进度 |
| 处理结果 | 明确结论 | 已解决/转升级/待备件 |

## 六、最佳实践建议（5条）

### 1. 数据及时更新

> 系统价值取决于数据质量。养成当天事当天记录的习惯，避免事后补录导致的信息遗漏。

- 客户拜访后24小时内录入跟进记录
- 项目节点完成后立即更新状态
- 商机有进展时立即更新阶段

### 2. 善用筛选和搜索

> 系统数据量会持续增长，学会使用筛选和搜索可以大幅提升工作效率。

- 收藏常用的筛选条件组合
- 使用关键词模糊搜索
- 导出筛选后的数据做二次分析

### 3. 关注数据质量

> 脏数据会影响报表准确性，录入时注意数据规范。

- 客户名称使用标准全称
- 日期字段避免使用"大约""待定"
- 金额字段统一为人民币元
- 状态字段及时更新，避免长期停留在"进行中"

### 4. 协作与共享

> CRM是团队协作工具，信息共享能提升整体效率。

- 及时上传有价值的知识素材
- 客户拜访记录详细填写，便于同事接手
- 项目文档上传到项目附件
- 给有价值的素材点赞，帮助优质内容被发现

### 5. 安全与保密

> 客户数据是核心资产，请注意信息安全。

- 妥善保管账号密码，不共享账号
- 离开工位时退出系统
- 不导出敏感数据到个人设备
- 发现数据异常及时报告管理员
